# ElectroMarket Analytics — Final Report

## Project Goal

The goal of this project is to analyze sales, customer behavior, product performance and marketing efficiency for a fictional electronics marketplace.

## Dataset

The dataset was synthetically generated using Python.

It includes:

- customers
- products
- orders
- order items
- web sessions
- reviews

## Tools Used

- Python
- pandas
- SQLite
- SQL
- matplotlib
- seaborn
- GitHub

## Key Metrics

The project analyzes the following metrics:

- Revenue
- Profit
- Average Order Value
- Repeat Purchase Rate
- Cancellation Rate
- Return Rate
- Conversion Rate
- Customer Segments

## Key Findings

1. Smartphones and Laptops generated the highest revenue.
2. Accessories had lower revenue but strong profit potential.
3. Paid Ads generated many customers, but their retention was weaker.
4. Referral and Email channels showed better customer quality.
5. A small group of loyal customers generated a large share of revenue.
6. The main customer retention problem appeared after the first purchase.

## Business Recommendations

1. Increase investment in Referral and Email channels.
2. Launch retention campaigns for At Risk customers.
3. Promote accessories as cross-sell products after smartphone and laptop purchases.
4. Investigate cancellation and return reasons.
5. Build a loyalty program for repeat customers.
